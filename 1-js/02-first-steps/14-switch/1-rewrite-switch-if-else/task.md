# "स्विच" को "इफ" में बदलें

`if..else` का उपयोग करके कोड लिखें जो निम्नलिखित `स्विच` के अनुरूप होगा:

```js
switch (browser) {
  case 'Edge':
    alert( "You've got the Edge!" );
    break;

  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    alert( 'Okay we support these browsers too' );
    break;

  default:
    alert( 'We hope that this page looks ok!' );
}
```
